# BlogApplication

A simple blog application built with Spring Boot and Spring Data JPA, using MySQL as the database.

## Features

- Create, read, update, and delete blog posts
- Add a picture to a blog
- Pagination
- Comment on blog posts
- RESTful API

## Technologies Used

- Spring Boot
- Spring Data JPA
- MySQL
- Maven

## Prerequisites

- Java 17 or higher
- Maven 3.6.0 or higher
- MySQL 8.0 or higher
